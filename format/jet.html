Jet¶

Store information about jets. The following selection is applied:

  • \(p_T \geq 5\) GeV

All the branches for this analyzer are stored inside std::vector<> of the corresping type listed below.

int refpdgid¶

The PDG id of the gen jet associated to the jet

float refdrjt¶

\(\Delta R\) between the jet and its gen jet

float refarea¶

Jet area of the gen jet

int8_t partonFlavor¶
int8_t hadronFlavor¶

Flavor of the jet. See this twiki page for a detailed explanation

float jtarea¶

Jet area of the get

float jtjec¶

Jet Energy Correction factor applied to this jet. Scale the impulsion of the jet with the inverse of this value to get the raw jet

float beta¶
float betaStar¶
float betaClassic¶
float betaStarClassic¶

Compute \(\beta\) and \(\beta^\star\), given by:

\[\begin{split}\beta &= \frac{\sum{p_T^{\text{CH, PV}}}}{\sum{p_T^{\text{CH}}}} \\ \beta^\star &= \frac{\sum{p_T^{\text{CH, PU}}}}{\sum{p_T^{\text{CH}}}}\end{split}\]

where the sum is over all the charged (\(\text{CH}\)) constituents of the jet, \(\text{PV}\) stands for constituents coming from the primary vertex and \(\text{PU}\) for constituents coming from another vertex.

Classic values are computed using MiniAOD fromPV() function (See here for more details):

  • \(\text{PV}\) are constituents with fromPV() == 3
  • \(\text{PU}\) are constituents with fromPV() == 0

Other values are computed by iterating over all the vertices and finding which one the constituent comes from, requiring dZ() < 0.2.

float dZ¶

dZ of the constituent with the biggest \(p_T\)

float DRweighted¶
\[\begin{split}&= \frac{\sum{p_T^2} \Delta R^2}{\sum{p_T^2}}\end{split}\]

where the sum if over all the constituents of the jet

float fRing0¶
float fRing1¶
float fRing2¶
float fRing3¶
float fRing4¶
float fRing5¶
float fRing6¶
float fRing7¶
float fRing8¶

Fraction of transverse momentum of the jet in a annulus in \(\Delta R\) plane between the jet and the constituents. The annulus is given by the number at the end of the branch name. For fRing6, it means \(0.6 < \Delta R < 0.7\), etc.

Note

fRing8 contains the fraction of transverse momentum for \(\Delta R \geq 0.8\)

uint16_t nCh¶
uint16_t nNeutrals¶

Number of charged and neutrals constituents of the jet

float ptD¶
\[\begin{split}&= \frac{\sqrt{\sum{p_T^2}}}{\sum{p_T}}\end{split}\]

where the sum if over all the constituents of the jet

float PUJetId_fullDiscriminant¶
int PUJetId_cutBasedId¶
int PUJetId_fullId¶

Store information about PU jet ID. See this twiki page for more detail.

float QGTagger_qgLikelihood¶

Quark / Gluon tagger discriminant

b-tagging¶

Additionnaly, a set of branches are created dynamically, one for each b-tagging algorithm available. The name of these branches are by definition not known in advance because the available b-tagging algorithm may change. However, all these branches name ends with BJetsTag so they are easily identifiable.

float *BJetsTag¶

b-tagging algorithm discriminant. The name of the branch is the actual name of the b-tagging algorithm.
