Lepton¶

Both electrons and muons share a common set of branches related to isolation computation. These branches are documented below, and specific branches are documented separately for Muon and Electron.

Common branches¶

float chargedHadronIsoR03¶
float chargedHadronIsoR04¶

Isolation for charged hadrons in various cone sizes (\(R = 0.3\) or \(R = 0.4\))

float neutralHadronIsoR03¶
float neutralHadronIsoR04¶

Isolation for neutral hadrons in various cone sizes (\(R = 0.3\) or \(R = 0.4\))

float photonIsoR03¶
float photonIsoR04¶

Isolation for photons in various cone sizes (\(R = 0.3\) or \(R = 0.4\))

float puChargedHadronIsoR03¶
float puCHargedHadronIsoR04¶

Isolation for PU charged hadrons in various cone sizes (\(R = 0.3\) or \(R = 0.4\))

float relativeIsoR03¶
float relativeIsoR04¶

Relative isolation of the particle in various cone sizes (\(R = 0.3\) or \(R = 0.4\))

\[\begin{split}I_\text{rel} &= \frac{ I^\text{CH} + I^\text{NH} + I^\text{Ph} }{ p_T }\end{split}\]
float relativeIsoR03_deltaBeta¶
float relativeIsoR04_deltaBeta¶

Relative isolation of the particle in various cone sizes (\(R = 0.3\) or \(R = 0.4\)), with delta beta correction

\[\begin{split}I_\text{rel} &= \frac{ I^\text{CH} + \max{\left(I^\text{NH} + I^\text{Ph} - 0.5 I^\text{CH, PU}, 0 \right) } }{ p_T }\end{split}\]
float relativeIsoR03_withEA¶
float relativeIsoR04_withEA¶

Relative isolation of the particle in various cone sizes (\(R = 0.3\) or \(R = 0.4\)), with effective area correction

\[\begin{split}I_\text{rel} &= \frac{ I^\text{CH} + \max{\left(I^\text{NH} + I^\text{Ph} - \rho \times \text{EA}, 0 \right) } }{ p_T }\end{split}\]

where \(EA\) is the effective area. The values used are extracted on PHYS14 samples and summarised on these slides.
